# Le Quadrant — Déploiement Vercel

## Option 1 — Drag & Drop (le plus simple, 2 minutes)
1. Aller sur https://vercel.com → créer un compte gratuit
2. Cliquer "Add New Project"
3. Glisser-déposer ce dossier `lequadrant`
4. Cliquer Deploy → URL publique générée !

## Option 2 — Via terminal
```bash
npm install -g vercel
cd lequadrant
vercel
```

## Fichiers
- index.html → HTML principal
- styles.css → Styles & animations
- app.js → JavaScript (réservation, pages, PDF)
- vercel.json → Config Vercel
